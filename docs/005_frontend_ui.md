# 005 フロントエンド UI 実装

## 概要

Vue コンポーネントを実装し、バックエンド API と連携した TODO 管理画面を完成させる。

## 画面仕様

- TODO の一覧表示
- テキスト入力による新規追加
- チェックボックスによる完了／未完了のトグル
- 削除ボタンによる一件削除

## ToDo

- [ ] `src/App.vue` を実装する
  - マウント時に `GET /api/todos` を呼び出して一覧を取得・表示する
  - 入力フォームと送信ボタンを配置する
  - TODO リストを `TodoItem` コンポーネントに渡す
- [ ] `src/components/TodoItem.vue` を実装する
  - `todo` を props で受け取る
  - チェックボックス変更時に `PATCH /api/todos/{id}` を呼び出す
  - 削除ボタン押下時に `DELETE /api/todos/{id}` を呼び出す
  - 操作後に親コンポーネントへイベントを emit して一覧を再取得する
- [ ] 新規追加フォームから `POST /api/todos` を呼び出し、成功後に一覧を更新する
- [ ] 完了済み TODO にスタイル（打ち消し線など）を適用する
- [ ] API エラー時にユーザーへフィードバックを表示する
